# Docs Archive

This directory is non-normative.

Rules:
- Files here are historical context only.
- Do not treat archive files as current behavioral or implementation authority.
- Canonical docs live at:
  - `docs/intent-ssot-v1.md` (intent)
  - source docstrings in:
    - `src/storage/storage-port.ts`
    - `src/storage/sqlite-schema.ts`
    - `src/storage/sqlite-storage.ts`
    - `src/phase3-extension.ts`
    - `.pi/live-drive/scm-live-drive.ts`
  - `docs/generated/implementation-reference.md` is generated from those docstrings and is convenience output only.

Contents:
- `spec-legacy/` — superseded spec/SSOT files, including pre-docstring implementation markdown snapshots.
- `build-notes-legacy/` — historical rebuild notes and reports.
- `session-notes/` — archived in-progress notes, handoff prompts, and checkpoint summaries.
- `write-down/` — dated checkpoint notes captured during implementation sessions.
